About
	MushRaider Brigde provides a mechanism for sharing users between drupal and MushRaider v1.4 .

Settings
    Permissions
    	In order to grant access to share login the "mushraiderbridge authenticate" permission have to be set for all roles.

    MushRaider configuration side
	    Define "Secret key".
	    Define "Third party url" to http:/yourDrupalWebsite/mushraiderbridge/auth.

	Drupal bridge configuration side
	    Set same value in "Secret key".
